## 💡 技术笔记

1. **Vben Admin 路由**: 通过 `import.meta.glob('./modules/**/*.ts')` 自动加载
2. **i18n**: 使用 `$t('page.quant.xxx')` 调用翻译
3. **组件懒加载**: `() => import('#/views/...')` 按需加载
4. **API 层**: 使用 `requestClient` 统一封装 HTTP 请求
